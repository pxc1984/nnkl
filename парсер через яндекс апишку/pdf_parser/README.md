# PDF Parser — Парсер документов с неявной структуры

Парсер преобразует PDF-документы с неявной/разорванной структурой в чистый Markdown, используя LLM (Qwen3 235B через Yandex AI Studio).

## Стек

| Компонент | Библиотека | Назначение |
|-----------|-----------|------------|
| PDF-извлечение | `pymupdf` | Извлечение текста и рендеринг страниц в изображения |
| Обработка изображений | `Pillow` | Масштабирование, конвертация в base64 |
| HTTP-клиент | `httpx` | Асинхронные запросы к Yandex AI Studio API |
| Retry-логика | `tenacity` | Автоматические повторы при ошибках API |
| Конфигурация | `pydantic` + `python-dotenv` | Валидация настроек из .env |
| CLI | `rich` + `argparse` | Красивый вывод в терминал |

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

Скопируйте `.env.example` в `.env` и укажите ваши ключи:

```bash
cp .env.example .env
# Отредактируйте .env
```

```env
YANDEX_API_KEY=your_api_key_here
YANDEX_FOLDER_ID=your_folder_id_here
LLM_MODEL=qwen3:235b
MAX_TOKENS=32000
TEMPERATURE=0.1
```

## Использование

### Базовый режим (постраничная обработка с изображениями)
```bash
python main.py document.pdf -o result.md
```

### Только текст (быстрее, без изображений)
```bash
python main.py document.pdf --text-only -o result.md
```

### Batch-режим (весь документ одним запросом)
```bash
python main.py document.pdf --batch -o result.md
```

### Python API
```python
from parser import PDFParser

parser = PDFParser()
markdown = parser.parse("document.pdf")
print(markdown)
```

## Режимы работы

| Режим | Скорость | Качество | Стоимость | Когда использовать |
|-------|----------|----------|-----------|-------------------|
| Постраничный + изображения | Медленно | Высочайшее | Высокая | Сложные макеты, таблицы, колонки |
| Постраничный, текст | Средне | Высокое | Средняя | Простые документы с читаемым текстом |
| Batch, текст | Быстро | Хорошее | Низкая | Короткие документы (< 20 стр) |
